# Enviroment Variables - Ortam Değişkenleri

- Sadece belirli ortama özgü (geliştirme / test / yayınlama) tanımlanan değişkeneler denir.
- Geliştrime sürecinde herkesle paylaşmak istemeyeceğimiz hassas veya değişken verilerde kullanırız
- `VITE_DEĞİŞKEN_İSMİ=DEĞER` formatında tanımlanır
- `import.meta.env.VITE_DEĞİŞKEN_İSMİ` formatında erişilir
